# Mangalam HDPE PIPES Web Page

A responsive product page for Mangalam HDPE pipes, built with HTML, CSS, and JavaScript.

## Features
- Fully responsive design (up to 1600px, down to 360px mobile).
- SVG-based exact logo recreation.
- Clean layout using Flexbox and Grid.
- Interactive Modals (Datasheet download).
- Accordion FAQs.
